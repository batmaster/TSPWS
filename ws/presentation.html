How to implement


WS -> login system -> sign in , sign out, sign up
Sign in
POST /signin
Content-type: application/json
{
"username" : "xxx",
"password" : md5(pass)
}

Sign out
POST /signout
{
"username" : "xxx"
}

Sign up
POST /signup
{
"username" : "xxx",
"password" : md5(pass),
"firstname" : "first name",
"lastname" : "last name"
}


-> inventory system -> add product , edit product , delete product , search product
add product
POST /product
Content-type: application/json
{
"productName" : "yyy",
...
}

edit product
PUT /product/{id}
{
"productName" : "yyy",
...
}

delete product
DELETE /product/{id}

search product
GET /product?str=xxx
Accepted: application/json

-> shopping -> add to cart , increase/delete quantity , search product
add to cart
POST /shopping/{id}/{quantity}

increase/delete quantity
POST /shopping/{id}/{quantity}

search product
GET /product?str=xxx
Accepted: application/json
